# Breast Cancer Classification

Machine learning models used:
- Logistic Regression - %97.66

## Major Steps
1 - Preparing the data
2 - Visualizing the data
3 - Data filtering
4 - Model Implementation
5 - Model Prediction
6 - Deploy Model
